About

Our aim is to reduce the amount of computing experience a user requires when using our systems at JD Hotels.

We recognised the difficulty in learning the architecture of the company and how to navigate and use our system.
To eliminate the need for users with computing experience, to simplify the system and make more user friendly we have converted the system into a Chatbot.
By converting our system into a Chatbot users will only be required to communicate with our system in their native language, as if they were communicating with a human.

Rules

This is a working prototpye, and still under development. So could you please follow the below rules when using the chatbot.

  • Please DO NOT use any punctuation when making your enquiry!
  • Please DO NOT use any ellison! E.g. Who's, Haven't, They're etc.
  • Please SUBMIT your feedback after each query using YES if you are satisfied with your results NO if not.
